# The Aviary

Three bird games. One favorite. You decide.

**Visit:** https://kevin-sotka.github.io/the-aviary/

| Game | One line | Play |
|---|---|---|
| [Murmuration](https://github.com/kevin-sotka/murmuration) | Paint the dusk sky with a starling flock | https://kevin-sotka.github.io/murmuration/ |
| [Dawn Chorus](https://github.com/kevin-sotka/dawn-chorus) | Echo birdsong, wake a watercolor forest | https://kevin-sotka.github.io/dawn-chorus/ |
| [Halcyon](https://github.com/kevin-sotka/halcyon) | The kingfisher's refraction-perfect dive | https://kevin-sotka.github.io/halcyon/ |

## Voting
Voting is one tap, right on the hub page. Tallies are stored in Firebase (Realtime Database), one vote per browser, and you can change your vote any time by tapping a different game. The pinned GitHub issues remain, but only as discussion threads now, not the ballot.

## How these were made
Nine concepts entered a three-round design tournament judged on beauty ceiling, first-ten-seconds fun, bird-truth, thumb-friendliness, and buildability. These three survived. Each is a single dependency-free `index.html`: canvas art and synthesized WebAudio only, nothing downloaded. Built by the SodClaw Game Foundry.
